A farming simulation game inspired by Hay Day, built with **Phaser 3** and **TypeScript**.

## Features
- Isometric farm grid with crop planting & harvesting
- 5 crop types: Wheat, Corn, Carrot, Tomato, Strawberry
- Level progression & XP system
- Order board for fulfilling deliveries
- Silo inventory management
- "Sun-Soaked" skeuomorphic UI design

## Getting Started

```bash
cd hayday-clone
npm install
npm run dev
```

Open `http://localhost:5173` in your browser.

## Controls
- **Left-click** a tile to plant/harvest
- **Right-click drag** to pan the camera
- **Scroll** to zoom in/out
